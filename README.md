# Multi-Tier Hospital Application on AWS

A Hospital Management System deployed on AWS using a secure multi-tier architecture with Terraform.

## Overview

This project demonstrates deployment of a Flask web application on AWS EC2 with Amazon RDS as the database. Infrastructure was provisioned using Terraform.

## Architecture

- EC2 for application hosting
- RDS for database
- VPC with public and private subnets
- IAM for access management
- Security Groups for controlled traffic

## Technologies Used

- AWS EC2
- AWS RDS
- AWS VPC
- AWS IAM
- Terraform
- Python Flask
- Linux
- Git
- GitHub

## Key Features

- Multi-tier architecture
- Infrastructure as Code using Terraform
- Secure networking with VPC and Security Groups
- Application deployment on EC2
- Managed database integration with RDS
- Scalable cloud infrastructure

## Deployment

```bash
git clone https://github.com/vkkprathik/multi-tier-hospital-aws-app.git
cd multi-tier-hospital-aws-app
cd terraform
terraform init
terraform plan
terraform apply
